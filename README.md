# Cumulus-BGP-Lab
Lab to play with running BGP on Cumulus.

## Setup ##
1.) Add Cumulus to your local system:
 `vagrant plugin install vagrant-cumulus`
 `vagrant box add cumuluscommunity/cumulus-vx`
