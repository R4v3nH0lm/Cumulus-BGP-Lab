## Requirements
1. VirtualBox (https://www.virtualbox.org/wiki/Downloads)
2. Vagrant (https://www.vagrantup.com/downloads.html)
3. Git (https://git-scm.com/downloads)

## Setup
1. Clone the repo.
```bash
git clone https://github.com/R4v3nH0lm/Cumulus-BGP-Lab.git
```

## Usage

### In terminal
1. Bring up the lab.
```bash
vagrant up.
```
